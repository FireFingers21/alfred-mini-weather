## Setup

**Note**: This workflow requires [jq](https://jqlang.github.io/jq/) to function, which comes preinstalled on macOS 15 Sequoia and later.

## Usage

View local weather information with the `wt` keyword. Select between hourly or daily forecasts.

![Using the wt keyword](Workflow/images/about/keyword.png)

![Viewing Hourly Forecast](Workflow/images/about/hourly.png)

![Viewing Daily Forecast](Workflow/images/about/daily.png)

Append `::` to the configured [Keyword](https://www.alfredapp.com/help/workflows/inputs/keyword) to access troubleshooting actions, such as clearing the workflow's cache.

![Other actions](Workflow/images/about/otherActions.png)

Configure the [Hotkeys](https://www.alfredapp.com/help/workflows/triggers/hotkey/) as shortcuts for viewing each forecast type.
